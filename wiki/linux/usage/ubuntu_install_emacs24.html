2 问题解决

  1. 提示"No X toolkit could be found"的解决

解决:

sudo apt-get install libgtk2.0-dev
  1. 提示"configure: error: The following required libraries were not found: libXpm libgif/libungif libtiff"

解决:

sudo apt-get install libxpm-dev
sudo apt-get install libjpeg62-dev
sudo apt-get install libgif-dev
sudo apt-get install libtiff4-dev
sudo apt-get install libncurses5-dev
  1. 系统原有的ctags在安装完emacs24后可能会执行出错

原因: emacs安装了自己的ctags, 如果不用可以把它去掉, 或者改名.
