Linux通过命令设置屏幕分辨率

date: 2013.05.23

目录:

  • 1 背景
  • 2 查看分辨率
  • 3 设置分辨率

1 背景

没有桌面系统的时候, 比如使用awesome, 屏幕显示分辨率不对就需要用命令行来设置.

2 查看分辨率

调用命令:

xrandr

会出现类似如下结果:

default connected 1280x800+0+0 0mm x 0mm表示当前默认是1280x800.

3 设置分辨率

输入命令:

xrandr --output default --mode "1920x1080"

即可变为1920x1080的分辨率。
